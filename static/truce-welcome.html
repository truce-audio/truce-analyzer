This installer installs one or more example plugins that ship with the truce audio plug-in framework. It is not the truce framework itself — that's a Rust crate you build against. The examples are here so plugin authors can compare host behaviour, study reference layouts, and verify their toolchain.

Pick the formats your DAW understands on the next screen — every format is checked by default, and you can drop any you don't need. Plug-ins land under /Library/Audio/Plug-Ins/ when you install for all users, or under ~/Library/Audio/Plug-Ins/ when you install for yourself only. Restart your DAW after install to pick up the new plug-in.
